Pangeo Gallery¶

Welcome to the Pangeo Gallery website. This site allows you to browse different Pangeo use cases. The site is organized into galleries, listed below, containing one or more notebooks. Each gallery is hosted in a standalone GitHub repository. If you’re interested in contributing a new gallery, please see the Contributor Guide.

Gallery for CESM LENS on AWS

A gallery of notebooks developed to demonstrate analysis of CESM LENS data publicly available on Amazon S3 (us-west-2 region) using xarray and dask

Pangeo & Dask Gateway.

How to use Dask Gateway on Pangeo Hubs and Binders for scalable computing.

Cloud Storage Benchmarks

Investigation of the throughput of various cloud storage formats and services. Prepared for the 2020 EarthCube Meeting by Ryan Abernathey.

Landsat 8 Tutorial

Use Pangeo platform to efficiently analyze a stack of Landsat8 imagery on AWS

Pangeo Tutorial Gallery

An gallery of notebooks to introduce the Pangeo software stack.

CMIP6 Gallery

Examples of analysis of Google Cloud CMIP6 data using Pangeo tools.

Example Gallery

An example gallery of notebooks used for debugging the Pangeo Gallery infrastructure.

Physical Oceanography

Examples of how to use Xarray, Dask, and Holoviews to load, process, and visualize cloud-based ocean data. Datasets include satellite observations (gridded sea-surface height) and ocean model output (POP, MOM6).

ESIP Gallery

An gallery of notebooks developed by ESIP Collaborators

Gallery for XGCM Examples and Usecases

Examples and tutorials using xgcm with models and observational datasets.
